# CortexAI Sentinel

## Hybrid AI-Powered Disaster Intelligence & Emergency Response Infrastructure

---

# Overview

CortexAI Sentinel is a hybrid AI-powered disaster intelligence platform designed to assist civilians during emergencies such as fires, floods, cyclones, gas leaks, and communication blackouts.

The platform combines:

- Hybrid Online/Offline AI
- Emergency Analytics
- GPS Navigation
- Disaster Heatmaps
- SOS Broadcasting
- Rescue Coordination
- Voice Intelligence
- AI Image Analysis
- Safe-Zone Detection

inside a futuristic emergency-response dashboard.

# Gemma 2B Integration

CortexAI Sentinel uses Gemma 2B running locally through Ollama as the core offline emergency-intelligence engine.

Gemma 2B enables:
- low-latency AI reasoning
- offline emergency guidance
- survival recommendation generation
- disaster-response intelligence

This allows CortexAI Sentinel to continue functioning during internet outages and infrastructure collapse.

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/CortexAI-Sentinel.git
cd CortexAI-Sentinel
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```
